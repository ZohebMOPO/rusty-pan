# Rusty-pan

> v0.0.2

Rusty pan is a `pansexual` themed unix shell.

## How to run

You need to have rust installed in your system.

Clone the repository:

```
git clone https://github.com/ZohebMOPO/rusty-pan/blob/main/README.md && cd rusty-pan
```

Build the project:

```
cargo build
```

Run the project:

```
cargo run
```
